Minimax and Tabular Q-Learning for Tic Tac Toe and Connect 4
This repository contains the implementation of Minimax and Tabular Q-Learning algorithms for playing the games of Tic Tac Toe and Connect 4. The project is developed by Tingzhou Wan, a student at Trinity College Dublin.

Introduction
The repository includes the following algorithms:

Minimax with alpha-beta pruning
Tabular Q-Learning
These algorithms are implemented in Python and are designed to be run in Google Colab.

Getting Started
To run the code, follow these steps:

Go to Google Colab
Sign in with your Google account
Click on "File" > "Upload Notebook" > "Choose File" and select the .ipynb file from this repository
Once the notebook is loaded, you can run the cells one by one or click on "Runtime" > "Run all" to execute the whole notebook
